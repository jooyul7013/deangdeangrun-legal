서비스는 자체 회원가입 절차 없이 **카카오 계정 OAuth(소셜 로그인)** 만을 통해 가입·로그인이 이루어집니다. 카카오에서 다음 항목을 제공받습니다.

| --- | --- | --- |
| 필수 | 카카오 회원번호(`kakao_id`) | 동일 사용자 식별 목적 |
| 선택 | 이메일 주소 | 이용자가 카카오 동의 화면에서 동의한 경우에 한해 수집. 미동의 시 `kakao_<회원번호>@kakao.nodomain` 형태의 가상 이메일을 내부 식별자로 자동 생성 |

| 선택 | 프로필 이미지 URL | 카카오 프로필에서 제공받음 |

수집 항목과 동의 범위는 이용자가 카카오 로그인 동의 화면에서 직접 확인 후 동의합니다.

### 1-3. 위치정보 수집·처리 정책 (특별 고지)

- 본 서비스는 「위치정보의 보호 및 이용 등에 관한 법률」상 **위치정보를 수집·이용**합니다.
- 수집 시점: 이용자가 러닝 세션을 시작하여 **러닝 포그라운드 서비스가 실행 중일 때에 한정**합니다.
- 백그라운드(앱이 종료되거나 러닝 세션이 중단된 상태)에서는 위치를 수집하지 않습니다. 이를 위해 Android `ACCESS_BACKGROUND_LOCATION` 권한을 **선언하지 않습니다.**
- 러닝 세션이 진행 중인 동안에는 시스템 알림(러닝 추적 알림)이 항상 표시되며, 이를 통해 이용자는 위치 수집 상태를 인지할 수 있습니다.
- 수집되는 위치정보의 정확도는 Android 의 `ACCESS_FINE_LOCATION`(정밀) 또는 `ACCESS_COARSE_LOCATION`(대략) 권한 부여 여부에 따릅니다.

## 3. 개인정보의 보유·이용 기간

회사는 원칙적으로 이용자의 개인정보를 **회원 탈퇴 시까지** 보유하며, 탈퇴 시 즉시 파기합니다. 단, 관계 법령에 따라 일정 기간 보존이 필요한 경우 해당 기간 동안 분리 보관합니다.

| 항목 | 보유 기간 |
| --- | --- |
| 회원 식별 정보(카카오 회원번호, 이메일, 닉네임, 프로필 이미지) | 회원 탈퇴 시까지 |
| 러닝 기록(`run_records` 테이블) | 회원 탈퇴 시까지. 단, 이용자가 앱 내에서 개별 기록을 삭제하면 즉시 파기 |
| 인증 토큰(액세스/리프레시 토큰) | 토큰 만료 또는 로그아웃 시까지 (단말기 내부) |

> **법정 보존 의무**: **해당 없음**. 본 서비스는 무료 서비스이며 재화·용역 결제 및 청약철회 절차가 존재하지 않으므로 「전자상거래 등에서의 소비자보호에 관한 법률」 등의 법정 보존 의무가 발생하지 않습니다.

- 러닝 진행 중 포그라운드 알림 표시를 위해 Android 13(API 33) 이상에서 알림 권한이 필요합니다. 거부 시에도 러닝 자체는 동작하나, 시스템 권장 사항상 허용을 권장합니다.

## 9. 개인정보의 안전성 확보 조치

회사는 다음과 같은 기술적·관리적 조치를 시행합니다.

- **전송 구간 암호화**: 모든 클라이언트–서버 통신은 TLS(HTTPS) 로 암호화
- **저장 구간 암호화**: Supabase 매니지드 PostgreSQL 의 디스크 암호화(at-rest encryption)
- **접근 통제(RLS)**: PostgreSQL Row Level Security 정책을 통해 본인의 `profiles` 와 `run_records` 만 접근 가능
- **인증 토큰 관리**: Supabase JWT 토큰은 만료 시점에 단말기에서 무효화/갱신
- **최소 권한 수집**: 백그라운드 위치 권한 미선언, 광고 식별자 미수집
